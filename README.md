# craigpesme.github.io

Static academic website for Craig Pesme.

## Deployment

Upload or commit the contents of this folder to the root of the `craigpesme.github.io` repository. GitHub Pages will serve `index.html` automatically from the repository root.

No build step is required.

## Structure

- `index.html` - main website page
- `assets/css/styles.css` - site styling
- `assets/js/main.js` - small navigation script
- `assets/img/` - portrait, favicon and social preview image
- `assets/files/` - CV and paper PDFs
- `.nojekyll` - tells GitHub Pages not to run Jekyll processing

## Updating papers or CV

Replace the relevant PDF in `assets/files/` and keep the filename the same so the website links continue to work.
